Your browser can finally talk with local TCP.

Hardware Bridge Required

To talk to local printers, you need to install a one-time bridge component.

1
Download and unzip the bridge package.
2
Run install_setup_mac.command (or Windows .bat).
3
Restart Chrome to activate the bridge.
Need to remove it? Simply run the uninstall file in the same folder.
