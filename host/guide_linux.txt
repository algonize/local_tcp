Local TCP Bridge - Linux Setup
-----------------------------------------

1. Open a terminal in this folder.
2. Run: `bash install_setup_linux.sh`
3. The "Smart Setup" will auto-detect or install Node.js (Ubuntu/Fedora/Arch).
4. IMPORTANT: Restart Chrome after the installation finishes.

-----------------------------------------
🗑️ UNINSTALLATION
-----------------------------------------
Should you need to remove the bridge from your Linux system:
1. Open a terminal.
2. Run: bash uninstall_setup_linux.sh
3. This will remove the files and the Chrome registration.
