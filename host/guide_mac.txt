Local TCP Bridge - MacOS Setup (Terminal)
-----------------------------------------

To avoid security blocks, follow these exact steps:

1. Open 'Terminal' on your Mac (Cmd + Space, then type 'Terminal').
2. Type exactly: bash 
   (Note: Add a SPACE after 'bash')
3. DRAG the file 'install_setup_mac.sh' into the terminal window.
4. Press ENTER.

Success! You will see a green checkmark message.
Restart Chrome to complete.

-----------------------------------------
🗑️ UNINSTALLATION
-----------------------------------------
1. Open Terminal.
2. Type: bash 
3. DRAG the file 'uninstall_setup_mac.sh' into the terminal.
4. Press ENTER.
