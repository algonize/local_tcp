Local TCP Bridge - MacOS Setup
-----------------------------------------

1. Locate the 'install_setup_mac.pkg' file.
2. RIGHT-CLICK the file and select 'OPEN'.
3. A popup will appear. Click the 'OPEN' button.
4. Follow the setup wizard to finish.
5. IMPORTANT: Restart Chrome (Cmd+Q) after installation.

🛡️ WHY RIGHT-CLICK?
-----------------------------------------
Since this is an unsigned package for your local hardware, macOS 
initially blocks it. Using "Right-Click > Open" is the official 
Apple shortcut to trust the installer immediately.

-----------------------------------------
🗑️ UNINSTALLATION
-----------------------------------------
Should you need to remove the bridge from your Mac:
1. Locate the 'uninstall_setup_mac.pkg' file.
2. RIGHT-CLICK the file and select 'OPEN'.
3. Follow the steps to completely remove the bridge.
